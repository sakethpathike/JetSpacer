<p><i>JetSpacer</i> pulls data from multiple NASA APIs and wraps it all in a minimal UI.</p><p><br><b>Features:</b></p><ul><li>Custom pagination for APOD Archive, Mars Gallery, and Headlines.</li><li>Search APOD, Mars Rover Photos, and NASA Image Library.</li><li>EPIC shows latest Earth imagery.</li><li>Bookmarking for APOD and Mars Rover Photos.</li><li>Caching support for Headlines with manual clear option.</li><li>Default API key included (or use your own: <a href='https://api.nasa.gov/#signUp' target='_blank' rel='nofollow noopener'>NASA API Signup</a>).</li></ul>
